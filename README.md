# Telegram Reminder Bot

Асинхронний бот-планувальник завдань з інтерактивним інтерфейсом та збереженням даних у SQLite.

## Функціональні можливості
* Керування планами через Inline-меню (календар на поточний тиждень).
* Збереження, перегляд та очищення завдань по днях.
* Автоматична розсилка нагадувань про плани на день (щоранку о 09:00).
* Робота з базою даних для збереження стану користувачів.

## Технологічний стек
* Python 3.10
* aiogram 3.x (Asynchronous framework)
* SQLite3
* Render (Cloud Deployment)

## Архітектура та розгортання
Проект спроектований для роботи в хмарному середовищі. Змінні оточення (BOT_TOKEN) конфігуруються через файл .env. Для стабільної роботи реалізовано автоматичне ініціювання бази даних при першому запуску.

## Встановлення та запуск
1. Клонувати репозиторій.
2. Встановити залежності:
   pip install -r requirements.txt
3. Налаштувати BOT_TOKEN у змінних оточення.
4. Запустити скрипт:
   python bot.py
