# 📅 Telegram Planner Bot

Асинхронний Telegram-бот для планування щоденних справ.

## 🚀 Функції
- 🗓 Вибір днів через Inline-меню.
- 💾 Збереження планів у базу даних SQLite.
- 🔔 Автоматичні ранкові нагадування (щодня о 09:00).
- ☁️ Розгорнутий на Render (Web Service).

## 🛠 Технології
- Python, Aiogram 3, SQLite, Aiohttp.

## ⚙️ Встановлення
1. Клонувати репозиторій.
2. Встановити залежності: `pip install -r requirements.txt`.
3. Створити `.env` файл із токеном бота.
4. Запустити: `python bot.py`.
